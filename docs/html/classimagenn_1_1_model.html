◆ train()

double imagenn::Model::train ( const std::vector< SpatialExample > & data,
double speed,
bool verbose = false,
double clip_value = 5.0,
bool use_clip = true )

Обучает модель на наборе примеров за один проход.

Порядок примеров перемешивается на каждом проходе: иначе данные, отсортированные по классам, приводят к катастрофическому забыванию и предсказанию одного класса.

Аргументы
[in]dataобучающие примеры
[in]speedскорость обучения
[in]verboseвыводить ли потерю по каждому примеру
[in]clip_valueпорог ограничения дельты веса
[in]use_clipприменять ли ограничение
Возвращает
суммарная потеря по всем примерам
