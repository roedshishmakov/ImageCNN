Подробное описание

Слой нейронов.

Владеет своими нейронами, нейроном смещения и связями смещения.

std::size_t size,
const ActivationBase & activation,
bool use_bias )

Создаёт слой заданного размера.

Веса смещения инициализируются нулём: случайные смещения при активном bias заталкивают часть ReLU-нейронов в отрицательную зону и выключают их.

Аргументы
[in]typeтип нейронов слоя
[in]sizeколичество нейронов
[in]activationфункция активации нейронов
[in]use_biasдобавлять ли нейрон смещения
